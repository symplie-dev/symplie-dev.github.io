NoteDown is a lightweight notepad powered by Markdown

Create Rich Notes

NoteDown lets you create beautiful notes that are visually consistent. This makes going back to reread them easier and more enjoyable.

NoteDown supports GitHub Flavored Markdown so you can embed code snippets with syntax highlighting, tasks, tables and more.

Common Note Taking Elements with a Click

H

NoteDown is perfect for developers and writers who are familiar with Markdown. But if you're not a Markdown pro, that's ok too! NoteDown provides helper buttons for creating common note taking elements like headers, lists, tasks, links, block quotes and code blocks.
